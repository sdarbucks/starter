01. 확장 프로그램
- Korean Language Pack for VS Code : 한글화
- Beautify : 코드 자동 정리
- Auto Rename Tag : 태그 이름을 한 번에 변경
- Live Server : 브라우저 출력

02. 단축키
- Ctrl + Alt + L : Beautify 기능
- Ctrl + Alt + S : 폴더 전체 저장
- Shift + Tab : 왼쪽으로 내어쓰기
- Alt + Shift + ↓ : 한 줄 아래에 똑같은 행 복사
- F12 : 브라우저 개발자 도구
- Ctrl + Shift + P : 메뉴 버튼 찾기

03. 팁
- [Code -> 기본 설정 -> 바로 가기 키] : 명령창 열기
- link라고 입력하고 <Tab>키를 누르면 태그 자동 완성

04. 전체 파일의 들여쓰기 공백문자를 2칸으로 설정하기
- Ctrl + Shift + P : 메뉴 버튼 찾기
- [기본 설정: 사용자 설정 열기]
- 상단 검색창에 tab size를 입력
- 공백 문자 4칸 -> 2칸으로 변경

05. 태그
- <title> 태그 : HTML 문서 제목
- <link> 태그 : CSS 파일 연결
- <script> 태그 : JS 파일 연결
- <input type="text" /> 태그 : 검색창
- <input type="checkbox" /> 태그 : 체크 버튼

06. 화면에 이미지 출력하기
- <F12>키 : 개발자 도구
- [Elements] 탭 -> 맨 왼쪽 선택(Select) 아이콘 클릭

07. 주의사항
- index.html 파일은 절대 폴더 안으로 이동 X
- 항상 전체 프로젝트의 최상위(Root) 경로에 위치

08. GIT 푸시
- $ git add .
- $ git commit -m "사이트 설명 추가"
- $ git push origin master

09. Codepen.io
- 새로운 코드를 간편하게 작성하고 테스트하는 사이트

10. 태그의 속성과 값
- <태그 속성="값"></태그>
- <img src="./cat.jpg" alt="고양이" />
- src : source 약자
- alt : alternative 약자

11. 글자와 상자
- 글자(인라인 요소) 예) <span> 태그
- 상자(블록 요소) 예) <div> 태그

12. 여백
- 외부 여백 : margin 속성
- 내부 여백 : padding 속성
